Our Project
Mental health Prediction Using Machine Learning
The main objective of this project is to build a machine learning model to accurately predict whether or not employees the in the dataset have recommend of treatment or not .
Details about the dataset
This dataset is originally from the National Institute of IT Technology. The objective of the dataset is to diagnostically predict whether or not a employee has mental health, based on certain diagnostic measurements included in the dataset.

Modules in our Project:
  • Exploratory Data Analysis : It is an approch to analyse the explotary data with visual techninices it used to trends or patterns with help of the statsical summary and graphically.
  • Data Preprocessing : Preprocessing in data mining technique which used to transform the raw data in a useful and efficient manner.
  • Feature Engineering : Process of selecting,manipulating,and transforming raw data into features that can be used in supervised learning.
  • One Hot Encoding : It is a common way of Preprocessing categorical features for machine learning models.
  • Base Models : A baseline model is essentically a simple model that acts as a reference in machine learning project .
  • Model Tuning : Tuning is the process of maximizing a model's performance without overfitting or creating too high of a variance.
  • Comparison of Final Models : It is compared on basics of Time complexity, Space complexity, Sample complexity,Parametricity.
  • Reporting : when a client is reported ,their case is first reviewed by our automated Disciplinary System.
